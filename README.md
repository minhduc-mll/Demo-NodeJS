## Prerequisites

- Require `Node version 18.x`

## Installing

To install, follow these steps:

First , clone project

```
git clone https://github.com/minhduc-mll/Product-management-api.git
```

Create .env file and add environment variables

```
PORT = 
MONGODB_URL = 
JWT_TOKEN = random
CLOUDINARY_NAME = 
CLOUDINARY_KEY = 
CLOUDINARY_SECRET = 
```

Then run the command

```
npm install
npm run dev
```
